# NodeJS-Swagger-API-Using-Swagger-JSDoc
Swagger is a set of open-source tools built around the OpenAPI Specification that can help you design, build, document and consume REST APIs.

### Install Packages
```
yarn
```

### Run
```
node index.js
```
